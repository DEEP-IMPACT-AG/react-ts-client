### Install dependencies
```
npm install
```

### Runs the app in the development mode
Build and open your browser to http://localhost:8888.
```
npm run dev
```

### Linting JS & CSS
```
npm run lint
```

### Builds and runs the app in production
```
npm run prod
```

### Start server
```
npm run start
```
